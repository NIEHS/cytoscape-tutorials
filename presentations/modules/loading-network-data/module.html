Loading Network Data

  • There are many methods for creating networks in Cytoscape:
    • Importing pre-existing, unformatted text or Excel files.
    • Importing pre-existing, formatted network files.
    • Importing networks from Public Databases
    • Creating an empty network and manually adding nodes and edges.
    • Using CyREST via R, Python, etc or command line scripts.

Loading Network Data

  • Cytoscape can import network data from:
    • Structured Files (or URLs)
      • XGMML: eXtensible Graph Markup and Modelling Language
      • SBML: Systems Biology Markup Language
      • BioPAX
      • PSI-MI
      • SIF: Simple Interaction Format
      • GML: Graph Markup Language
      • ... and others depending on loaded Apps
    • Unstructured Files
      • Excel, TSV, CSV
