Workshop Goals

By the end of this workshop you should be able to:

  • Understand the major applications of network biology
  • Navigate Cytoscape's interface
  • Import data into Cytoscape
  • Find relevant networks and pathways
  • Perform basic topological and other network analyses
  • Manipulate network layouts and data visualization
  • Know where to find additional Cytoscape apps and tutorials
